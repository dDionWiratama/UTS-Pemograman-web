TENTANG SAYA

Nama saya Dion Wiratama. Biasanya, semua orang memanggil saya Dion. Saya lahir di Madiun, 08 Agustus 2004. Saya saat ini berumur 19 tahun. Saya saat ini kuliah di Politeknik Negeri Madiun. Sebelum menempuh jenjang perkuliahannya, saya bersekolah di SDN SUKOREJO 03, SMPN 1 MEJAYAN, dan SMAN 1 MEJAYAN Sewaktu bersekolah, saya sangat menyukai mata pelajaran Fisika. Tidak hanya itu saja, saya juga menyukai beberapa olahraga bela diri, seperti Taekwondo, Pencak Silat, dan Karate. Hobi saya sehari-hari adalah membaca buku. Saya juga suka bermain olahraga bulutangkis di akhir pekan. Saya memiliki banyak teman, baik itu teman sekelas sewaktu sekolah dulu, perkuliahan, organisasi, maupun teman magang. Meskipun demikian, saya hanya bercerita kepada teman terdekat saya, yaitu Andi dan Budi.
